# Purpose and tech stack

## Purpose
Starter kit for building an in-app checkout experience with Paddle. Package: `@paddle/in-app-checkout-starter` (private).

## Tech stack
- **Framework**: Next.js 15 (Turbopack for dev), React 19
- **Paddle**: @paddle/paddle-js
- **UI**: Radix UI, Tailwind CSS 4, lucide-react, sonner, next-themes
- **Package manager**: pnpm
- **Lint**: ESLint (Next config), TypeScript

## Structure
- `src/` – app source
- `public/` – static assets
- `next.config.ts`, `tsconfig.json`, `postcss.config.mjs`, `components.json` (shadcn)
