# Suggested commands

- `pnpm install` – install dependencies
- `pnpm dev` – next dev with Turbopack
- `pnpm build` – next build
- `pnpm start` – next start (production)
- `pnpm lint` – next lint
- `pnpm tsc` – type check (noEmit)
- Copy `.env.example` to `.env` and set Paddle env vars for local dev.
